Bobandtanner Software develops mobile apps. Currently, we have 3 mobile apps:

  • Square Feet Calculation Utility (iOS only)
  • Triangle Help (iOS only)
  • Algebra Tool Mobile (iOS and Android)

Square Feet Calculation Utility is by far the most popular. As of February 1, 2021, a new version of Square Feet Calculation Utility is under development. The old version, 1.1, was developed in 2016. It is crashing on the new iOS, 14. The new version of Square Feet Calculation Utility is written completely from scratch and will be different (and hopefully better) than the old version.

Triangle Help is a triangle solver which was developed in 2016 following Square Feet Calculation Utility. It does not calculate area and it is not designed to complement Square Feet Calculation Utility.

Algebra Tool is a human-driven computer algegra system. It lets the human make decisions but it does not allow mistakes. It is difficult to use so there is a tutorial video about it. A website version of Algebra Tool is available here: https://winkelmantanner.github.io/algebratool/.
