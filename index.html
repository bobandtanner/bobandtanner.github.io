Bobandtanner Software develops mobile apps. Currently, we have 3 mobile apps:

  • Square Feet Calculation Utility (iOS only)
  • Triangle Help (iOS only)
  • Algebra Tool (web, iOS, and Android)

Square Feet Calculation Utility is by far the most popular. This app was first developed and published in 2016, when Tanner was just 17 years old. The original app, version 1.1, crashes on iOS 14.2. The new version, 1.2, is written completely from scratch and is very different from the old version. Tanner began developing version 1.2 in January 2021 in response to crashes on iOS 14.2. Version 1.2 was released in February 2021.

Triangle Help is a triangle solver which was developed in 2016 following Square Feet Calculation Utility. It does not calculate area and it is not designed to complement Square Feet Calculation Utility.

Algebra Tool is a human-driven computer algegra system. It lets the human make decisions but it does not allow mistakes. It is difficult to use so there is a tutorial video about it. A website version of Algebra Tool is available here: https://winkelmantanner.github.io/algebratool/.

Square Feet Calculation Utility

Square Feet Calculation Utility is a specialized calculator for floor area in square feet. The user enters numerous rectangles in feet and inches and the app calculates the area. A web version of the app is available at https://squarefeetcalculator.neocities.org, but it is much cheaper than the iOS app.

Unlike our other apps, this app is heavily used. As of January 2021, it makes about 50 app units per day. It has received a few reviews on the app store, which have been very positive. After this app had been on the store for 5 years, iOS 14.2 came out and this app crashes immediately when opened on iOS 14.2. We received feedback from a user regarding the app crashing on the new iOS. So Tanner developed version 1.2, which was released on February 7, 2021.
